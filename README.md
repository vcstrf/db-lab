# Лабораторная работа №1

Лабораторная работа №1 по базам данных. Тема БД: гитары.

## Запросы

1. **Поиск (по названию, фирме, типу, форме)**

   **Тип**: `GET`

   **URL**: `/search/`

2. **Список предложений на товар**

   **Тип**: `GET`

   **URL**: `/product/offers/id`

3. **Список характеристик товара**

   **Тип**: `GET`

   **URL**: `/product/attributes/id`

## Развертывание

В корневой папке проекта:

```bash
docker-compose up
```
Сайт развернется на [http://127.0.0.1:8080/](http://127.0.0.1:8080/)
